鼠标手势方案简介

通过鼠标手势可以有效提高日常电脑办公效率，此方案供大家参考，方案相应的配置文件在我的博文中的链接下载。

本鼠标手势方案针对Windows系统及常用软件的日常操作所定制的一套相对完整和有参考性的鼠标手势操作方案，针对全局通用和特定程序进行了优化，同时还支持同一手势在不同区域执行不同动作，较大提升其实用价值。

基于 StrokesPlus 制作，因其集成了Lua脚本引擎，可以通过简单的脚本实现灵活和强大的自定义鼠标手势，同时还支持Windows8系统，所以是 StrokeIt 很好的替代方案。

StrokesPlus 下载：http://www.strokesplus.com/downloads/
语言包下载：http://www.strokesplus.com/forum/topic/273/chinese


**应用**
将本目录中的 *.lua、*.xml 覆盖 StrokesPlus 根目录下的同名文件即可。
